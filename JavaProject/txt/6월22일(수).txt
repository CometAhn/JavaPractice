멤버변수 : 클래스 영역에 선언된 변수, 앞에 static이 붙은 변수를 클래스 변수.

지역변수 : 특정{ } 사이에서 선언한 변수로 { }사이에서만 사용 가능한 변수 - 특정지역 안에서만 사용이 가능하다.

인스턴트 변수 : 클래스영역에 선언되고 인스턴스 생성시 만들어진다.
		각각의 인스턴스마다 다른 값을 가질 수 있다.
		인스턴스 생성이 필수이며 각 인스턴스마다 다른 값을 가져야 할 때 사용된다.

클래스 변수 : 인스턴스 변수 앞에 satic을 붙이기만 하면된다.
	      인스턴스 변수가 각각의 인스턴스마다 다른값을 가질 수 있는 것과 달리 값을 공유하는 변수이다.
	      클래스 로딩시 생성되며 클래스 이름, 변수이름 이렇게 인스턴스 생성 필요없이 바로 쓸 수 있다.

지역 변수 : 메서드 내에 선언되며 메서드 호출시 생성되고 메서드가 종료되면 사라진다.

매개 변수 :파라미터라고 불린다. 메서드에서 입력값을 받을 때가 있는데 그때 사용되는 변수를 매개변수라고 한다.
	   매개변수도 메서드 내에서 선언된 것으로 간주되므로 지역변수이다.
	   인자 값은 호출시 메서드 입력부의 넣는 값이며 이 값은 매개변수에 복사되어 대입된다.


데이터의 형변환 : 데이터의 타입을 변경
자동 타입 변환 : 작은 데이터를 큰 쪽으로 옮겨줄 수 있는 것
타입별 크기
정수 : byte(1) < short(2) < int(4) < long(8)
실수 : float(4) < double(8)

강제 타입 변환 : 데이터를 강제로 변환
		 강제 타입 변환에서 사용자로부터 받은 입력값을 변환할 때 값의 손실이 일어나지 않도록 주의해야 한다.
		 값이 보존할 수 있는지 체크할 필요가 있습니다.
		 사용자가 입력한 데이터값이 변환하고자 하는 타입의 범위에 들어갈 수 있는지 확인

	int -> byte
	int i = 128;

	if ((i < Byte.MIN_VALUE) || (i > Byte.MAX_VALUE)) {
		System.out.println("byte 타입으로 변환할 수 없습니다.");
		System.out.println("값을 다시 확인하세요.");
	} else {
		byte b = (byte) i;
		System.out.println(b);
	}
