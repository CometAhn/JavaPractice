배열 = 저장공간에 대한 크기를 지정.
Set = 크기를 지정, 데이터량에 따라 크기를 늘어나거나 줄어든다.
		중복된 데이터를 가지지 않는다. 중복된 요소를 추가하면 false 반환.
		저장순서를 유지하지 않는다.
hashSet : 중복이 없고, 저장하는 순서가 없다.
hashSet() // 클래스의 기본 생성자
HashSet(Collection collection) // 컬렉션의 요소로 HashSet 객체생성
HashSet(int capacity) // capacity의 용량을 가진 객체 생성.

add() : 데이터를 추가 할 때 중복된 요소가 있다면 false을 반환. 요소가 추가되지 않는다.

TreeSet : set 인터페이스에서 졍렬 기능이 추가된 형태. 

List : 데이터를 일렬로 늘어놓은 구조.
		입력한 순서대로 요소가 유지되며, 중복을 허용.
		ArrayList, LinkdList
add("a") :a데이터를 리스트 맨 뒤에 추가.
add(1, "a") : 리스트 1+1 위치에 a 데이터를 넣는다. 원래 있던 값들은 뒤로 밀려서 저장된다.
get(1) : 리스트 1+1위치에 데이터 값을 가져온다.
IndexOF("a") : a가 저장된 위치의 값을 가져온다.
remove(1) : 리스트 1+1 위치에 있는 값을 제거한다. 뒤에 있는 값들이 앞으로 옮겨져서 저장된다.
set(1, b) 리스트 1+1위치에 값을 b로 변경한다.
ListIterator listiterator() : ListIterator 객체를 반환. 

Queue first in First out
Stack List in First out
